## Run Locally

Clone the project

```bash
  git clone https://github.com/chadirabii/github-searcher
```

Go to the project directory

```bash
  cd my-github-searcher
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
